## Quick start
Copy your development license file to `env/devel/cmsbs-conf`:
```bash
cp cmsbs.license env/devel/cmsbs-conf/
```

Install and run Universal Messenger:
```bash
gradle setup run
```

Login: http://localhost:8080/cmsbs (`admin` / `admin`)
